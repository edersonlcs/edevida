Como usar no Telegram (passo a passo)

          
    
            
  1. Abra seu bot no Telegram e envie /start.
    2. 
            
  2. Para ver ajuda rápida, envie /help.
    3. 
            
  3. Para abrir este painel, envie /painel.
    4. 
            
  4. Para resumo completo do dia, envie /resumo.
    5. 
            
  5. Para visão do corpo inteiro (5 níveis), envie /corpo.
    6. 
            
  6. Para ver foco de exames, envie /exames.
    7. 
            
  7. Para conversar sem registrar refeição, use /chat sua pergunta.
    8. 
            
  8. Para registrar alimentação, envie texto, foto ou áudio da refeição.
    9. 
            
  9. Para água, inclua no texto algo como: "bebi 500 ml de água".
    10. 
            
  10. No Telegram agora o fluxo é por rascunho: envie refeição/foto/áudio, ajuste se precisar, e confirme em Registrar refeicao.
    11. 
            
  11. Botões diários principais: Resumo de hoje, Status do corpo, Registrar refeicao, Sugestao proximo refeicao, Plano de hoje, Falar com IA, Painel e Help.
    12. 
            
  12. Tudo que for registrado no Telegram aparece aqui no painel.
